Atlas Express Website

📌 About the Project

This project is a modern landing page for a logistics company Atlas Express.
The website presents services, delivery calculator, tracking system, and contact information.

The goal of the project is to create a clean, responsive, and user-friendly website for cargo delivery services.

---

🚀 Features

- 📦 Delivery cost calculator
- 🔍 Shipment tracking system (demo)
- 🗺 Office location map (OpenStreetMap)
- 💬 WhatsApp contact integration
- 📱 Fully responsive design (mobile-friendly)
- ⭐ Customer reviews section
- 📰 Blog section

---

🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla JS)

---

📂 Project Structure

- "index.html" — main website file
- All styles and scripts are included inside the file
